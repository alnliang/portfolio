Pong

AP Computer Science Applications(AP CSA)

30 September 2019-11 October 2019

In this project, I worked on an Android application with two other students. We decided to implement a Pong game into our application. There were multiple screens on our application. There was a login screen, which had fields for both the username and password, a settings screen, which could change the color of the background, a victory screen, a defeat screen, as well as a screen to display the game itself. The Pong game itself was coded with Java, and animations were made using Java's draw function. The settings page was accessed through a three-dots menu. To win, the player must hit the ball five times, otherwise, they lose. One major difficulty my group and I had was on deciding how we were going to code a Pong game, since we were not aware of the draw function before the assignment of the project. I did the majority of the coding, while my partners did the majority of the presentation. Thus, I learned from this project about the draw function, as well as the UI of Android Studio rather thoroughly. All documentation and code can be viewed on Github.

Image Manipulation

Computer Science Principles(CSP)

4 March 2019-24 March 2019

In this project, my partner and I used Python to create a program that modifies images. Our program made collages out of the images given by the user, with an option for the user to add a logo on each image. We decided to create a collage function since we thought it was unique. Since there were many elements to the collage, we needed to figure out how to shrink the images, and which algorithms would work best to determine the width and height of the collage. During this project, my partner and I didn't communicate as well, since he wasn't online for most of the day. In the end, it seemed like I did most of the work, so another skill I learned was to manage my time efficiently, since I didn't have much time to work on the project, and my partner wasn't helping me. As a product of this ordeal, I have learned to manage others, as well. We had documents where we wrote daily logs and showed each function and its final product.
