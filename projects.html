Scratch Project

Computer Science Principles(CSP)

30 August 2018-6 September 2018

This project was used as a way to test our understanding of Scratch. The objective of the project was to create a program that ran without flaws with the use of Scratch. My partner and I decided to make a platformer with gravity physics and different levels. Although simple, the project displays our understanding of Scratch, as gravity physics are very complicated. We made sure the game ran smoothly by constantly debugging and testing out the code. During the project, I learned to think more logically, as this project was one of my first major encounters with programming. I also learned how to communicate more effectively, as the goal we were trying to reach wasn't easy. We also constantly documented our activity in a digital notebook. We had daily logs.



Appinventor Project

Computer Science Principles(CSP)

18 September 2018-9 October 2018

In this project, my partner and I used a tool called Appinventor to create a mobile application. We decided to create a multi-fuctional application. The app had 3 functions: A timer, Calendar/Reminder, and a NoteTaker. We decided to create a multi-tool because it's functional and simple. Despite the simplicity of the idea, the development of the application displayed our understanding of Appinventor. Since each function required different elements of Appinventor, we were able to utilize many of the tools provided, such as TinyDB and Clocks. We had daily logs where we documented what we worked on every day during the project.



Image Manipulation

Computer Science Principles(CSP)

4 March 2019-24 March 2019

In this project, my partner and I used Python to create a program that modifies images. Our program made collages out of the images given by the user, with an option for the user to add a logo on each image. We decided to create a collage function since we thought it was unique. Since there were many elements to the collage, we needed to figure out how to shrink the images, and which algorithms would work best to determine the width and height of the collage. During this project, my partner and I didn't communicate as well, since he wasn't online for most of the day. In the end, it seemed like I did most of the work, so another skill I learned was to manage my time efficiently, since I didn't have much time to work on the project, and my partner wasn't helping me. As a product of this ordeal, I have learned to manage others, as well. We had documents where we wrote daily logs and showed each function and its final product.
